# Console Library Manager

A simple console application for managing a library, created as a learning project while studying Java.

## Features

* Add new books
* Display all books
* Search books by title and author
* Delete books
* Mark books as read
* Display only unread books
* Input validation using exceptions

## Technologies

* Java 25
* ArrayList
* Object-Oriented Programming (OOP)
* Exception Handling
* Git & GitHub

## Project Structure

```
src/
├── Main.java // Application entry point
├── Book.java // Book model
├── BookManager.java // Library management logic
└──BookStats.java // Data Transfer Object
```
